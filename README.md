# とらのすけ - toranosuke

いろんな翻訳ツールで翻訳してくれるウェブサイト

## 対応翻訳ツール

### Google Translate API

https://github.com/iPolyomino/google_translate_api

## 動作環境

- Node.js 12.x
- yarn 1.22.x
- Google App Engine
- Google Cloud Build

## インストール

```zsh
git clone https://github.com/iPolyomino/toranosuke.git
cd toranosuke
yarn
yarn dev
```
